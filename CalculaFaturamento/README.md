# Cálculo de Faturamento Diário

## Descrição

Este simples executável calcula e retorna informações sobre o faturamento diário de uma distribuidora ao longo de um ano. O programa realiza os seguintes cálculos:

1. O menor valor de faturamento ocorrido em um dia do ano.
2. O maior valor de faturamento ocorrido em um dia do ano.
3. O número de dias no ano em que o valor de faturamento diário foi superior à média anual.

## Instruções para Baixar e Executar

1. **Clone o Repositório**

   Clone o repositório contendo o código-fonte do projeto para o seu ambiente local. Utilize o seguinte comando:

   ```bash
   git clone https://github.com/mathieux-dev/TargetSistemas.git
   ```

2. **Compile e Execute o Programa**

   - **Abra o Projeto no Visual Studio** ou outra IDE de sua escolha.
   - **Compile e Execute** o programa a partir da IDE.

## Arquivo de Entrada

- O programa considera que o vetor com os valores de faturamento diário já está carregado no código. Você pode modificar o vetor no arquivo principal conforme necessário para os cálculos.
